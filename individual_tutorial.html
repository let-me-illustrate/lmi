Working with an individual case

First, do either of the following:

Create a new illustration

Load a saved illustration file

Next, fill in your data:

Change input for an illustration

Then press OK to calculate the illustration and see a summary.

Here are the things you can do next with this illustration:

Print an illustration

Save an illustration

While this illustration is displayed, you can also create or load another illustration for an individual or group case.

Creating a new illustration

  Press this button and pick Illustration.

Or pull down the File menu, pick New, and then pick Illustration.

Loading a saved illustration

  Press this button. Or pull down the File menu and pick Open.

Either way, go to Files of type: and choose Illustration. Pick the file you want and press Open.

You can also open files by dragging and dropping them on lmi.

Changing input for an illustration

After you’ve created a new illustration, or loaded a saved one, here’s how to modify it.

  Press this button.

Or pull down the Illustration menu and pick Edit cell.

Make your changes, then press OK to recalculate the illustration and refresh the summary.

Printing an illustration

  Press this button.

Or pull down the File menu and pick Print.

You can also use other print commands.

Saving an illustration

  Press this button.

Or pull down the File menu and pick Save.

Either way, type a file name and press Save.

To save a file under a new name or in a new location, pull down the File menu and pick Save as.
